TAF 2.11

Prerequisites:
1. .NET Framework 2.0 (could be installed with windows updates, found in network (dotnetfx.exe) or downloaded from http://www.microsoft.com/downloads/details.aspx?familyid=0856eacb-4362-4b0d-8edd-aab15c5e04f5&displaylang=en)
2. Sikuli X (could be installed from www.sikuli.org)

Installation Notes:
1. Copy all files to local folder.

How to run:
1. Open TAF.exe.config, specify path to test suite in 'TestSuite' setting and run TAF.exe. Additional information could be found in
	'Documentation' folder.
